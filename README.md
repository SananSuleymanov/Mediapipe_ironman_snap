# MediaPipe OpenCV Iron Man filter

This project aims to create the simple Iron Man filter using MediaPipe and OpenCV. 
The main steps of project as follows:
1. Detection of the nose landmark of face using MediaPipe
2. Calculation of the initial position of the face from camera frame for mapping PNG mask
3. Mapping mask

The main.py file contains all of the code needed in order to build targeted Iron Man filter.
